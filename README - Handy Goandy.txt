README - Handy Goandy

This is AR Project Created by Vuforia with Vuforia Database.

Unity Version: 2019.4.2f1
Vuforia Version: 9.2.8

if you found any issue for tracking you could use these 2 methods:
1. Create your own Vuforia database with different object.
2. Change the option on player and enemy target from database into image
   (Select Player/EnemyTarget > Image Target Behaviour > Type > From Image) Make sure you put new image in unity.
   
if you found any issue with unitypackage, please message me in handygoandy2@gmail.com.

Thank You.
